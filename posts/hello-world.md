## A Beautiful Equation

One of the most elegant equations in mathematics is Euler's identity:

$$e^{i\pi} + 1 = 0$$

This remarkable formula connects five fundamental mathematical constants: $e$, $i$, $\pi$, $1$, and $0$.

## The Quadratic Formula

When solving quadratic equations of the form $ax^2 + bx + c = 0$, we use:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

## Code and Mathematics

Here's a simple example of how we might implement mathematical concepts in code:

```javascript
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}
```

The Fibonacci sequence follows the recurrence relation:

$$F_n = F_{n-1} + F_{n-2}$$

with initial conditions $F_0 = 0$ and $F_1 = 1$.
